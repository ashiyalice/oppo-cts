# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-02-26

### Added
- Multi-language support added: Japanese (ja), Chinese Simplified (zh), and English (en).
- Detailed explanations added to each step of the Setup Wizard for all supported languages.

### Changed
- Setup Wizard UI updated to use a `ScrollView` for the instruction text area to prevent text cutoff on smaller screens due to longer, more detailed instructions.
